# QA Test Assignment

This repository contains QA artifacts created as part of a test assignment, demonstrating skills in designing and executing decision tables, test cases, checklists, and bug reports.

## Structure

- **Bug_Reports/** - Bug reports documentation
- **Checklists/** - Testing checklists
- **Decision_Tables/** - Decision tables for different scenarios
- **Test_Cases/** - Test cases for application pages
- **Specifications/** - Specifications for testing

## Test Assignment

### 1. Decision Tables

Calculate the discount amount for cinema tickets under the following conditions:

- student - 10%
- retiree - 25%
- birthday - free
- before 14:00 - 20%

Note: Different discounts can be combined.

### 2. Test Cases

Write test cases for `site/admin/create` page.

### 3. Checklist

Write a checklist for `site/admin/dashboard` page.

### 4. Bug Reporting

Describe 10 found bugs.
